# Hi, I'm Lazar Bukejlovic

I build web applications with a product mindset.

What interests me most is not making pages look finished on the surface, but building projects that feel like they belong in a real software environment — structured well, visually sharp, easy to understand, and backed by features that make practical sense together.

My work is centered around full-stack SaaS-style applications, business dashboards, workflow tools, and interfaces that feel deliberate rather than thrown together. I care about clean presentation, but I care just as much about whether the product logic underneath actually holds up.

---

## What I’m Building Toward

I’m focused on becoming a strong full-stack web developer with a portfolio built around serious, product-style web applications.

That means I’m not interested in filling my GitHub with random clones or disconnected tutorial projects. I’d rather build fewer projects with better structure, stronger use cases, cleaner execution, and a more believable product feel.

## How I Think About Projects

I like projects that feel coherent.

Not just “more features,” but features that actually belong together.

Not just “nice UI,” but UI that supports the product clearly.

Not just “backend added somewhere,” but backend logic that improves how the product behaves.

That is the standard I try to build around:
- clear structure
- practical workflows
- strong presentation
- believable product logic
- polished execution without unnecessary noise
